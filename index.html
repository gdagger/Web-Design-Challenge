Summary: Latitude vs. X

The purpose of this project was to analyze how distance from the equator impacts weather. In order to complete this analysis, we pulled data from the OpenWeatherMap API to assemble a dataset consisting of over 500 cities.

Once the datset was assembled, we examined the relationship between distance from the equator (latitude) and various weather factors including temperature, cloudiness, humidity, and wind speed. This website contains the source data and visualizations of the four main factors that were explored. Each of the main visualization pages also contains a brief description of any observed trends and correlations.
